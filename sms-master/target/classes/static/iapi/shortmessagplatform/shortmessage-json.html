短信平台接口

短信平台接口Json格式

请求说明

  • http请求方式: POST
  • /sendSmsByJson
  • Josn数据例子：
    {
    "head": {
    "transCode": "MSG_PLATFORM",
    "transType": "req"
    },
    "data":
    {
    "source" : " channel ",
    "comCode" : " comCode ",
    "phoneNumber" : "18694325420",
    "messageContent" : "你好啊 ",
    "paramCode" : "1",
    "systemCode" : "sendmessageByaLiyun"
    }
    }

入参说明

参数 类型 最大长度 必传(Y是，N否) 字段描述
source String 30 Y 渠道标识
comCode String 30 Y 机构标识
phoneNumber String 20 Y 需要发送短信的电话号码
messageContent String 100 Y 短信内容
paramCode String 30 Y 开关的key
systemCode String 30 Y 微服务代码

返回说明

{
"head": {
"transCode": "MSG_PLATFORM",
"transType": "req"
},
"data":
{
"Message":"OK",
"Code":"OK"
}
}

出参说明

参数 类型 最大长度 必传(Y是，N否) 说明
Code String 60 Y 状态码（OK：成功、其它：失败）
Message String 60 Y 状态描述
