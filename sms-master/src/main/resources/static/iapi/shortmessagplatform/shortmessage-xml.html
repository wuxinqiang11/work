短信平台接口

短信平台接口XML格式

请求说明

  • http请求方式: POST
  • http://10.1.27.122:8080/gateway/msgsend/MSGSEND/sendSms
  • XML数据例子：
    <?xml version="1.0" encoding="utf-8" ?>
    <messages>
    <userId>channelcode</userId>
    <password>96e79218965eb72c92a549dd5a330112</password>
    <sendTime>2018-07-26</sendTime>
    <businessType>SMS_139986767</businessType>
    <organization>0000</organization>
    <sequenceNumber>123456789</sequenceNumber>
    <contents>
    <recvMobile>18698151330</recvMobile>
    <messageContent>{ "name1":"张三","name2":"意外险","name3":"2018-05-01","name4":"200","name5":"1310021656201", }</messageContent>
    </contents>
    </messages>

入参说明

参数 类型 最大长度 必传(Y是，N否，S条件必传) 字段描述
userId String 30 Y 渠道标识
password String 30 Y 渠道密码
sendTime String 20 Y 发送时间
businessType String 100 Y 业务模板号
organization String 30 Y 组织机构
sequenceNumber String 30 Y 短信唯一码
recvMobile String 30 Y 手机号码
messageContent String S 短信内容
businessNo String S 业务号

返回说明

<?xml version="1.0" encoding="utf-8" ?>
<messagesResponse>
<result>001</result>
</messagesResponse>

出参说明

参数 类型 最大长度 必传(Y是，N否) 说明
Code String 60 Y 状态码（001：成功、其它：失败）
